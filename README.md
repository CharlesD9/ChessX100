# ChessX100


## 1. Install Node.js (LTS)
Visit the official Node.js site, download, and run the **Windows (MSI)** installer:
```
https://nodejs.org/en/download
```

---

## 2. Install Git for Windows
Download Git for Windows (standalone installer, **x64**).  
Git is used to download the project source code.
```
https://git-scm.com/download/win
```

---

## 3. Open a Command Prompt
Press **Windows Key + R**, type `cmd`, and press **Enter**.

---

## 4. Clone and run the project
In the Command Prompt, run the following commands **one at a time**.  
Only proceed if the previous command completes without errors.

```powershell
git clone git@github.com:CharlesD9/ChessX100.git
cd ChessX100
npm install
npm start
```

If the above completed you should see a message like "ChessX100 running at http://localhost:3000"

## 5. Point a browser to the App
